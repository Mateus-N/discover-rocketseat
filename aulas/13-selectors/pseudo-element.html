Pseudo-elements

Com pseudo-elements nós podemos adicionar elementos HTML pelo próprio CSS

::pseudo-element-name

Exemplos

::before
Adiciona algo antes do conteúdo
li::before {
content: "> ";
}

::after
Adiciona algo após o conteúdo
li::after {
content: ";";
}

fisrt-line
Adiciona algo após o conteúdo
li::fisrt-line {
font-weight: bold;
}
