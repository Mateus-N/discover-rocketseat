Layouts e evolução

Layout tem a ver com a maneira que os elementos estão distribuidos na tela

Normal flow

Ou flow layout é a maneira que os elementos `block` e `inline` ficam na página

table

É a maneira de tabelas onde os a tag `table` recebe um display `table` fazendo com que os elementos internos como `td` e `tr` possam ser usados para montar uma tabela

tabless

Uso das propriedades `float` e `clear` para que os elementos possam mudar de posição na tela

</div>
<div style="float: right">

</div>
<div style="clear: both">
normal
</div>

flexbox

A caixa se torna flex, fazendo com que os elementos internos possam receber melhor:

  • Alinhamento
  • Ordenação
  • Tamanhos flexíveis

display: flex
