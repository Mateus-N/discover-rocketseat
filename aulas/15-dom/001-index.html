DOM (Document Object Model)

  • É o HTML convertido para um objeto JavaScript
  • API que representa e interage com o HTML
  • Estrutura de dados do tipo árvore criada pelo browser
  • Propriedades e métodos

Para que?

  • JavaScript usa a DOM para se conectar ao HTML
  • Manipular o HTML com JavaScript
  • Você só programa pra WEB por que existe a DOM

getElementById()

Utiliza o ID do elemento

getElementsByClassName()

Utiliza a classe do elemento, resulta em um HTMLCollection

  • Elemento 1
  • Elemento 2
  • Elemento 3

getElementsByTagName()

Utiliza a Tag HTML, resulta em um HTMLCollection

querySelector()

Utiliza o seletor CSS, pega apenas o primeiro que encontrar na página

querySelectorAll()

Utiliza seletor CSS e devolve todos os elementos referente ao seletor, resulta em uma NodeList

Diferença

Na NodeList é possível ser feita um forEach

textContent

manipula o conteúdo, podendo trocar tudo ou apenas adionar algo

innerText

Troca o texto interno do elemento

innerHTML

Permite a adição de HTML ao código
