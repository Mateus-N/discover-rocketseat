Cores

Usamos CSS para alterar cores do nosso documento.

Tipos

  • * background-color (para caixas)
  • * color (para textos)
  • * border-color (para caixas)
  • * outros...

Valores

Podemos definir valores por

  • * palavra-chave (blue, transparent)
  • * hexadecimal (#990011)
  • * funções: rgb, rgba, hsl, hsla

CSS

element {
/* keyword values */
/* currentcolor = cor atual, mesmo que não colocar nada */
color: currentcolor;
/* <name-color> values */
color: red;
color: orange;
color: tan;
color: rebeccapurple;
/* <hex-color> values 0-F */
/* RED GREEN BLUE TRANPARENT */
color: #090;
color: #009900;
color: #090a;
color: #009900aa;
/* <rgb()> values 0-255 */
color: rgb(34, 12, 64);
color: rgba(34, 12, 64, 0.6);
color: rgb(34 12 64);
color: rgba(34 12 64 / 0.6)
color: rgb(34.0 12 64)
color: rgba(34.0 12 64 / 60%)
/* <hls()> values */
/* Hue - Saturation - Lumiance */
color: hsl(30, 100%, 50%)
color: hsla(30, 100%, 50%, 0.6)
color: hsl(30 100% 50%)
color: hsla(30 100% 50% / 0.6)
color: hsl(30.0 100% 50%)
color: hsla(30.0 100% 50% / 60%)
/* Global values */
color: inherit;
/* herda a cor do elemento anterior */
color: Initial;
/* Volta a cor inicial */
color: unset;
/* Cor não definida, pega a cor do contexto */
}
